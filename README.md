# Teslo API

1. Clonar el repositorio

```bash
git clone
```

2. Clonar el archivo .env.example y renombrarlo a .env

```bash
cp .env.example .env
```

3. Instalar las dependencias

```bash
npm install
```

4. Levantar el servidor

```bash
npm run start:dev
```

5. Levantar la base de datos

```bash
docker-compose up -d
```
